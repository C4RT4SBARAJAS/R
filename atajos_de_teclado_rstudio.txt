Aquí te muestro algunos de los principales atajos de teclado o keyboard shortcut para Rstudio. Bastante útiles si lo que buscas es aumentar tu productividad al momento de programar en esta interfaz. Estoy seguro que hasta el más experto no conoce todas estas opciones. Los atajos que aquí te mostraré son los que Rstudio tiene por defecto al momento de su instalación, pero si te parecen que son muy complejas o difíciles de memorizar tienes la opción de perzonalizarlas. 

Para que los atajos funcionen deberás presionar las teclas al mismo tiempo una seguida de otra según lo que se indique:

Ctrl + Enter = Corre o ejectuta el código.

Alt + enter = Corre tu código pero sin avanzar el cursor una linea abajo.

Ctrl + Shift + M =  %>% (etiqueta de pipa)

Alt + - = <-- (asignador)

Ctrl + F = buscar y remplazar (mac = ⌘F)

Ctrl + Shift + F = buscar archivos (de sintaxis por ejemplo) (mac = ⇧⌘F)

Ctrl + . = te busca una función dentro del código (mac = ⌃.)

Ctrl + Shift + A = introducir una nueva línea de código  (mac = ⇧⌘A)

Ctrl + D = eliminar una línea de código (mac = ⌘D)

Ctrl+F11/F12 (con la tecla de fn seguramente) para cambiar de pestaña lo mismo Alt + ctrl + flecha derecha o izquierda (mac = ⌃F11 o ⌃⌥←)

Ctrl+p = te lleva de un paréntesis a su propio final (...,...) (ma = ⌃P)

Alt + up/down: mueve el código seleccionado arriba o abajo (mac = ⌥↑ or ↓)

Alt + shift + up/down: copia el código seleccionado arriba o abajo (mac = ⌥⌘↑ or ↓)#alt + shift + up/down: copia el código seleccionado arriba o abajo (mac = ⌥⌘↑ or ↓)

Ctrl + + = acerca el zoom 

Ctrl + - aleja el zoom

Alt + drag = múltiples cursores

Ctrl + L = limpiar la consola  (mac = ⌃L)

Ctrl + Shift + 1 = la ventana se centra en la sintaxis y la maximiza (mac = ⌃⇧1)

Ctrl + Shift + 8 = la ventana se centra en el ambiente y lo maximiza (mac = ⌃⇧8)

Ctrl + Shift + 6 = la ventana se centra en los plots y los maximiza (mac = ⌃⇧6)

Ctrl + Shift + Alt + 0 = vuelta a la normalidad. 

Ctrl + Shift + flecha arriba = seleccionar todo el código de arriba hasta que encuentre una línea vacia (mac = ⌃⌥⇧↑)

Ctrl + Shift + flecha arriba + flecha arriba  = todo lo de arriba lo mismo con la flecha para abajo.

Ctrl + A = seleccionar todo el código (mac = ⌘A)

Ctrl + Alt + Shift + M = renombra variables en el propio script (mac = ⌥⇧⌘M)

Ctrl + 1 o 2 = para oscilar entre el código y la consola. (mac  = ⌃1 o ⌃2)

Ctrl + Alt + B = runea todo el código hasta donde estes seleccionando (mac =  ⌥⌘B)

Ctrl + Shift + R = poner una etiqueta (tidy) (mac = ⇧⌘R or ⌃⇧R)
